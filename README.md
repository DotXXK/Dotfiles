# Dotfiles For Hyprland :D

These are my own Dotfiles, you can steal it if you want :D

**Dependencies**

* Hyprland (obviusly)
* Waybar
* Rofi
* Awww
* Swaync
* Alacritty

**Optional**

* Fish 
* Fastfetch
* Starship (For the terminal prompt)

## Installation

* Arch Linux (Or arch based distros like Manjaro,Artix,endevouros and cachyos)

Dependencies: sudo pacman -S waybar hyprland rofi awww swaync alacritty
Optional Dependencies: 'sudo pacman -S '

* Debian ( 13(Trixie with backports) | 14(Forky/unstable) )

"If you want to install hyprland on debian 13 you need to install the package 'trixie-backports'"
